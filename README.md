# California Housing Price Prediction
An end-to-end Machine Learning web application built using Scikit-learn and Streamlit.

## Features
- Stratified Train-Test Split
- ColumnTransformer Pipeline
- Random Forest Regressor
- Modular Prediction Pipeline
- Streamlit Web Interface

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Project Structure
- artifacts/ (Saved model & preprocessing pipeline)
- src/ (Custom modules)
- app.py (Streamlit frontend)
